# Tattoo Website

A modern, responsive website for a tattoo business built with React and Vite.

## Features

- Modern React 19 with Vite build tool
- Responsive design with Tailwind CSS
- Contact forms and quote requests
- Gallery and blog sections
- Stripe integration for payments
- Firebase backend services

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Start development server:
   ```bash
   npm run dev
   ```

3. Build for production:
   ```bash
   npm run build
   ```

## Tech Stack

- React 19
- Vite
- Tailwind CSS
- Redux Toolkit
- React Router
- Stripe
- Firebase
